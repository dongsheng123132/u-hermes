🐧 DIY: U-Hermes Linux Boot USB

Open-source Ventoy workflow, modeled after u-claw-linux

Quick recipe

git clone https://github.com/dongsheng123132/u-hermes.git
cd u-hermes\linux

.\1-prepare-usb.ps1
.\2-download-iso.ps1
.\3-create-persistence.ps1
.\4-copy-to-usb.ps1

# Boot target PC from USB, then run:
# sudo bash /media/*/Ventoy/u-hermes-linux/setup-hermes.sh
