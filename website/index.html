常见问题

U-Claw 和 U-Hermes 什么区别？

U-Claw 面向通用 AI 场景和 IM 机器人（QQ/微信/飞书接入）。U-Hermes 面向编程和真·自主 Agent（执行 shell、操作浏览器、读写文件）。两个都装在同一张 U 盘里，按需切换。

换电脑能用吗？

可以。U 盘绑定账户体系，插哪台 Windows 电脑都是同一个账户和余额。

U 盘坏了怎么办？

联系微信 hecare888 用订单号换新 U 盘，余额会转移。

Mac / Linux 能用吗？

Windows 版原生支持 Win10/11。Mac 可通过 Parallels 虚拟机使用。Linux 用户推荐免费的 开源 Live USB 版。

杀毒软件拦截 / 没法运行？

启动器是无签名 Electron 应用，360 / 火绒 / Windows Defender 可能拦截。请在杀毒软件里把 U 盘上的 launcher/U-Hermes.exe 加白名单。我们在申请代码签名证书，下个版本会解决。

启动后端口被占用？

默认端口 8642 / 8648。如果占用，启动器会自动 +1 避让。如仍有问题，从托盘退出后重启。

退款政策？

非质量问题 7 天内可退（U 盘未使用）。已充值的 AI 余额不可退，但可继续在 cloud.u-claw.org 使用。

开源版和商业版区别？

Linux 开源版需自己刻盘、配置、自备 API Key。Windows 商业版开箱即用，含成品 U 盘 + 预装 AI 额度 + 一键图形界面 + 售后。核心 AI 能力两者一致（都基于 NousResearch Hermes Agent）。

为什么不直接下载软件？

U 盘既是便携载体，也是账户绑定凭证和售后凭证。如果你只想要软件本身，欢迎用开源 Linux 版。
