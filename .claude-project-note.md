## 禁区 🚫

**绝不能提交**下列内容（属于闭源商业版）：

- USB 指纹生成逻辑（PowerShell、SHA256 规则）
- 激活码签名验证（ED25519 公钥/私钥、dealer 系统）
- 虾盘云 API Key 派发对接
- `UCLAW_CLOUD_*`、`uclaw2026activate` 等内部常量
- Electron 启动器源码（属于 `../u-hermes-pro/`）
- 客户的 DeepSeek/OpenAI API Key 等凭证

提交前请 `grep -riE "fingerprint|dealer|activate.*secret|sk-[a-f0-9]{30}" website/ linux/ docs/` 检查。
